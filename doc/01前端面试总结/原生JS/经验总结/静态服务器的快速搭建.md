## 静态服务器的快速搭建

```js
// 安装 http-server 依赖
// npm i http-server -D
```

- 添加相关执行脚步

```json
{
  "name": "01.demo",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "serve": "http-server -a 127.0.0.1 -p 9999"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "devDependencies": {
    "http-server": "^14.1.1"
  }
}
```
