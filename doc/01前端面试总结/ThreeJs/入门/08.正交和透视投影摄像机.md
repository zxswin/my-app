## 正交和透视投影摄像机

```js
// Three.js为我们提供了两种摄像机：正交投影摄像机，透视投影透视摄像机。
// 物体距离摄像机越远，它就会被渲染的越小。
// 正交投影，所有立方体尺寸大小都一样，摄像机的距离对渲染没有任何影响。
```

- 透视投影摄像机 THREE.PerspectiveCamera

```js
// fov	fov表示视场，是在摄像机中能够看到的那部分场景，例如，人类有接近180度的视场，而有的鸟类却有接近360度视场。一般来说，游戏开发中视场设置为60-90最佳。推荐使用默认值 50

// aspect（长宽比）	这是渲染结果横向和纵向尺寸的比值。例如，我们把窗口当作输出界面，则设置为：
// window.innerWidth / window.innerHeight

// near（近面距离）	定义了从摄像机多近的距离开始渲染。通常这个值会设置的尽量小，使得摄像机可以看到所有物体。
// 推荐设置为 0.1

// far（远面距离）	定义了从摄像机位置能看多远。如果这个值设置的太小，那么远处的东西将不会被渲染。
// 如果设置的过大，则会影响性能。
// 推荐设置为 1000

// zoom（变焦）	该属性可以让你放大缩小场景，小于1缩小，大于1放大，负数的话上下会颠倒。
// 推荐设置为 1
```

- 正交投影摄像机 THREE.OrthographicCamera

```js
// left（左边界）	可视范围的左边界，如果这个值设置为-100，那么你会看不到比这个左边界更远的物体。

// right（右边界）	任何比右边界远的物体都不会被渲染。

// top（上边界）	可被渲染区域的最上边。

// bottom（下边界）	可被渲染区域的最下边。

// near（近面距离）	基于摄像机所处的位置，从这一点开始渲染。

// far（远面距离）	基于摄像机所处的位置，渲染场景到此为止。

// zoom（变焦）	该属性可以让你放大缩小场景，小于1缩小，大于1放大，负数的话上下会颠倒。
// 推荐设置为 1
```
