## 混合应用开发

```js
// Capacitor
// Capacitor 是一个来自 Ionic Team 的项目，通过提供跨多个平台运行的 API，开发者可以使用单个代码库构建原生 iOS、Android 和 PWA 应用。

// NativeScript
// NativeScript 使用已熟悉的 Web 技能为跨平台（真正的原生）移动应用提供支持。两者结合在一起是开发沉浸式移动体验的绝佳选择。
```
