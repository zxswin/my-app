## 简单状态管理起步使用

```js
// 如果你有一处需要被多个实例间共享的状态，可以简单地通过维护一份数据来实现共享：
// 现在我们有了唯一的数据来源，但是，调试将会变为噩梦。任何时间，我们应用中的任何部分，在任何数据改变后，都不会留下变更过的记录。
var sourceOfTruth = {};

var vmA = new Vue({
  data: sourceOfTruth,
});

var vmB = new Vue({
  data: sourceOfTruth,
});
```

```js
// 为了解决这个问题，我们采用一个简单的 store 模式：
var store = {
  debug: true,
  state: {
    message: 'Hello!',
  },
  setMessageAction(newValue) {
    if (this.debug) console.log('setMessageAction triggered with', newValue);
    this.state.message = newValue;
  },
  clearMessageAction() {
    if (this.debug) console.log('clearMessageAction triggered');
    this.state.message = '';
  },
};
```

```js
// 每个实例/组件仍然可以拥有和管理自己的私有状态：
var vmA = new Vue({
  data: {
    privateState: {},
    sharedState: store.state,
  },
});

var vmB = new Vue({
  data: {
    privateState: {},
    sharedState: store.state,
  },
});
```
