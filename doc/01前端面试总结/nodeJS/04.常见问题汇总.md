## async/await 捕获错误

- 直接在后面接 .catch()

```js
const getData = async () => {
  const res = await getData().catch((err) => {
    console.log(err);
  });
};
```

- 使用 try/catch

```js
const getData = async () => {
  try {
    const res = await getData();
  } catch (err) {
    console.log(err);
  }
};
```
