## 日期选择器

- 安装

```ts
// npm install --save react-datetime
// yarn add react-datetime
```

- 简单的使用案例

```ts
import Datetime from 'react-datetime';
import 'moment/locale/zh-cn';
import 'react-datetime/css/react-datetime.css';
import './style.scss';

const DTPicker = props => {
  /* 默认属性 */
  const defaultProps = {
    // input控件的相关属性
    inputProps: {
      placeholder: '请选择日期',
    },
    dateFormat: 'DD/MM/YYYY', // 日期格式化
    timeFormat: false, // 禁止选择时间
    // 不能选中周六日
    isValidDate: current => {
      return current.day() !== 0 && current.day() !== 6;
    },
  };

  return (
    <div className="UI-DTPicker">
      <Datetime className="UI-DTPicker__Datetime" {...defaultProps} {...props} />
    </div>
  );
};

export default DTPicker;
```

- 可以通过 ref 获取日期对象的引用

```tsx
<DateTime ref="datetime" />;
// 切换到年视图
this.refs.datetime.navigate('years');
```

- 多语言

```tsx
// 引入语言支持文件
import 'moment/locale/fr';

// 如果日期组件需要展示不同的语言
// 则可以通过locale属性进行配置
<Datetime locale="fr-ca" />
<Datetime locale="de" />
```

## 属性说明

- value

```ts
/**
 * 属性名:value
 * 类型:Date
 * 默认值: new Date()
 * 描述：代表日期选择器的选中值
 * 在受控组件中使用
 * 可以是日期字符串或日期对象类型
 */
```

- initialValue

```ts
/**
 * 属性名:initialValue
 * 类型:Date
 * 默认值: new Date()
 * 描述：代表日期选择器的选中值
 * 在非受控制组件中使用
 * 可以是日期字符串或日期对象类型
 */
```

- initialViewDate

```ts
/**
 * 属性名:initialViewDate
 * 类型:Date
 * 默认值: new Date()
 * 描述：定义日历面板中的可见日期
 * 可以是日期字符串或日期对象类型
 */
```

- initialViewMode

```ts
/**
 * 属性名:initialViewDate
 * 类型:字符串或数字
 * 默认值: 'days'
 * 描述：默认展示的日历面板视图
 * 可选值'years', 'months', 'days', 'time'
 */
```

- updateOnView

```ts
/**
 * 属性名:updateOnView
 * 类型:字符串
 * 默认值: 自动推断
 * 描述：会通过日期的格式自动推断是要展示日月年那种类型的日历面板
 */
```

- dateFormat

```ts
/**
 * 属性名:dateFormat
 * 类型:字符串 或 布尔值
 * 默认值: true
 * 描述：接受任何Moment.js的格式化日期
 * true的时候使用默认的当前地址格式化日期
 * false则会禁用日历面板
 *
 */
```

- timeFormat

```ts
/**
 * 属性名:timeFormat
 * 类型:字符串 或 布尔值
 * 默认值: true
 * 描述：格式化时间
 * true的时候使用默认的当前地址格式化日期
 * false则会禁用日历面板
 */
```

- input

```ts
/**
 * 属性名:input
 * 类型: 布尔值
 * 默认值: true
 * 描述：是否展示日期输入框
 */
```

- open

```ts
/**
 * 属性名:open
 * 类型: 布尔值
 * 默认值: null
 * 描述：设置是否展示或隐藏日期控件
 * 默认情况下为input获取焦点的时候展示 在外部点击的时候隐藏
 */
```

- locale

```ts
/**
 * 属性名:locale
 * 类型: 字符串
 * 默认值: null
 * 描述：设置本地时区
 * 需要下载对于的locale文件
 */
```

- utc

```ts
/**
 * 属性名:utc
 * 类型: 布尔值
 * 默认值: false
 * 描述：设置utc时区
 * 设置为true则显示为utc时间
 */
```

- displayTimeZone

```ts
/**
 * 属性名:displayTimeZone
 * 类型: 字符串
 * 默认值: null
 * 描述：设置展示的时区
 */
```

- onChange

```ts
/**
 * 属性名:onChange
 * 类型: function
 * 默认值: 空函数
 * 描述：日期改变的回调函数
 */
```

- onOpen

```ts
/**
 * 属性名:onOpen
 * 类型: function
 * 默认值: 空函数
 * 描述：打开日期选择器的回调函数
 */
```

- onClose

```ts
/**
 * 属性名:onClose
 * 类型: function
 * 默认值: 空函数
 * 描述：关闭日期选择器的回调函数
 */
```

- onNavigate

```ts
/**
 * 属性名:onNavigate
 * 类型: function
 * 默认值: 空函数
 * 描述：日历模式改变的时候触发的回调函数
 */
```

- onBeforeNavigate

```ts
/**
 * 属性名:onBeforeNavigate
 * 类型: function
 * 默认值: ( nextView, currentView, viewDate ) => nextView
 * 描述：在日历视图改变之前触发 返回下一个被切换的视图
 * 如果返回false则回停留在当前视图不进行切换
 */
```

- onNavigateBack

```ts
/**
 * 属性名:onBeforeNavigate
 * 类型: function
 * 默认值: 空函数
 * 描述：日历控件返回之前的日历视图的时候触发
 */
```

- onNavigateForward

```ts
/**
 * 属性名:onNavigateForward
 * 类型: function
 * 默认值: 空函数
 * 描述：日历视图进入下一个视图的时候触发
 */
```

- className

```ts
/**
 * 属性名:className
 * 类型: 字符串或数组
 * 默认值: ' '
 * 描述：外层容器的自定义类名
 */
```

- inputProps

```ts
/**
 * 属性名:inputProps
 * 类型:object
 * 默认值: undefined
 * 描述：给input组件添加额外的属性
 */
```

- isValidDate

```ts
/**
 * 属性名:isValidDate
 * 类型:function
 * 默认值: ()=> true
 * 描述：接受当前日期和选择日期作为参数
 * 返回true或者false来决定那些日期可以被选择
 */
```

- renderInput

```ts
/**
 * 属性名:renderInput
 * 类型:function
 * 默认值: undefined
 * 描述：渲染一个自定义的input组件
 * 回替换掉之前的input组件
 */
```

- renderView

```ts
/**
 * 属性名:renderView
 * 类型:function
 * 默认值: (viewMode, renderDefault) => renderDefault()
 * 描述：渲染一个自定义的日历面板
 * 回替换掉之前的日历面板
 */
```

- renderDay

```ts
/**
 * 属性名:renderDay
 * 类型:function
 * 默认值: DOM.td(day)
 * 描述：渲染一个自定义的日期
 * 必须返回一个react组件
 */
```

- renderMonth

```ts
/**
 * 属性名:renderMonth
 * 类型:function
 * 默认值: DOM.td(month)
 * 描述：渲染一个自定义的月份
 * 必须返回一个react组件
 */
```

- renderYear

```ts
/**
 * 属性名:renderYear
 * 类型:function
 * 默认值: DOM.td(month)
 * 描述：渲染一个自定义的年份
 * 必须返回一个react组件
 */
```

- strictParsing

```ts
/**
 * 属性名:strictParsing
 * 类型:布尔值
 * 默认值: true
 * 描述：使用严格模式解析input
 */
```

- closeOnSelect

```ts
/**
 * 属性名:closeOnSelect
 * 类型:布尔值
 * 默认值: false
 * 描述：如果是true当选择一个日期的时候面板回关闭掉
 */
```

- closeOnTab

```ts
/**
 * 属性名:closeOnTab
 * 类型:布尔值
 * 默认值: false
 * 描述：当为true的时候input获取聚焦后按下tab键回关闭面板
 */
```

- timeConstraints

```ts
/**
 * 属性名:timeConstraints
 * 类型:object
 * 默认值: null 可传入 { hours: { min: 9, max: 15, step: 2 }}
 * 描述：设置输入时间的限制条件
 */
```

- closeOnClickOutside

```ts
/**
 * 属性名:closeOnClickOutside
 * 类型:布尔值
 * 默认值: true
 * 描述：点击外部是否关闭日历面板
 */
```
