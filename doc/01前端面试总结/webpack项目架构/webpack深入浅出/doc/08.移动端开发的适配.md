## 移动端开发屏幕适配方案.md

- 采用的方案

```bash
## 移动端的尺寸适配方案
npm install lib-flexible --save-dev
## 安装 postcss-pxtorem
npm install postcss-pxtorem --save-dev

## 在入口文件中引入
import 'lib-flexible';

## 在webpack配置文件中配置
// 解析less文件
{
  test: /\.less$/,
  use: [
    {
      loader: MiniCssExtractPlugin.loader,
      options: {
        esModule: true,
      },
    },
    {
      loader: 'css-loader',
      options: {
        sourceMap: true,
      },
    },
    {
      loader: 'postcss-loader',
      options: {
        sourceMap: true,
        ident: 'postcss',
        plugins: [
          require('postcss-preset-env')({
            browsers: 'last 5 versions',
            autoprefixer: { grid: true },
          }),
          require('postcss-pxtorem')({
            rootValue: 75,
            propList: ['*'],
          }),
        ],
      },
    },
    {
      loader: 'less-loader',
      options: {
        sourceMap: true,
      },
    },
  ],
  include: path.resolve(__dirname, 'src'),
},

```
