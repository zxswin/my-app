## Jest 配置

```js
// 1.package.json​来存储 Jest 的配置
// 2.可以通过​jest.config.js 来配置
{
  "name": "my-project",
  "jest": {
    "verbose": true
  }
}
// 3.或者通过 JavaScript：
// jest.config.js
//Sync object
module.exports = {
  verbose: true,
};
//Or async function
module.exports = async () => {
  return {
    verbose: true,
  };
};

// 请记住最后拿到的配置必须是可被 JSON 序列化的。
{
  "bail": 1,
  "verbose": true
}
```
